# Customer Retention Analysis - Python

## Tools Used
- Python, Pandas, Matplotlib, Seaborn

## Dataset
Download from Kaggle:
https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci

## Dataset
Online Retail Dataset - 541,909 transactions

## Key Findings
- 4,338 unique customers analyzed
- 30.4% Champions (best customers)
- 32.1% Lost Customers need re-engagement
- Average Order Value: $2,054
- Average Frequency: 4.3 orders

## Techniques Used
- RFM Analysis
- Customer Segmentation
- Data Cleaning
- Data Visualization
# Customer-Retention-Analysis
This project analyzes 541,909 transactions from an online retail store using Python. RFM analysis was applied to segment 4,338 customers into Champions, Loyal, At Risk and Lost groups. Key finding: 32.1% are Lost Customers while 30.4% are Champions. Results help businesses improve retention strategies.
